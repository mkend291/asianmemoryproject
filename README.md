## Quick Start

1. Open `index.html` in your browser
2. Navigate using the top menu

## Customizing Team Page

Replace placeholder images in `team.html`:
```html
<img src="path/to/your-image.jpg" alt="Team Member">
```

Update names, roles, and bios in the flip cards.
